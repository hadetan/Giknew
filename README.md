# Giknew (Phase 1 Skeleton)

Early scaffold based on `story.md`.

## Quick Start
1. Copy `.env.example` to `.env` and fill values.
2. Provide a Postgres instance and set `DATABASE_URL`.
3. Install deps and generate Prisma client.
4. Run dev server.
